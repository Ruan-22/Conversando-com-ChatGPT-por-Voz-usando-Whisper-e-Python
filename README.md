# Conversando com ChatGPT por Voz usando Whisper e Python

Código desenvolvido no Colab no Bootcamp da DIO

# 1. Gravação de Áudio
Usando Python e um pouco de JavaSript para gravar a voz e reproduzi-la\
O tempo padrão da gravação é de 5 segundos, porém podendo ser alterado na chamada da função

# 2. Reconhecimento de Fala com Whisper
Usando Whisper para transcrever o  texto gravado\
O modelo do Whisper pode ser mudado dependendo da necessidade

# 3. Integração com a API do ChatGPT
O texto trancrevido é enviado pro ChatGPT que poderá responder a uma possível pergunta feita

# 4. Sintetizando a Resposta do ChatGPT Como Voz
Usando o gTTS é possível converter a resposta dada pelo ChatGPT em fala
